# Netflex Clone (Frontend)

Simple Netflix-style landing page clone built with HTML and CSS only.

## Demo

Open `index.html` in your browser.

## Tech Stack

- HTML
- CSS

## Features

- Responsive layout
- Hero section with CTA
- Content rows and cards
- Footer links

## Project Structure

- `index.html`
- `style.css`
- `assets/`

## How to Run Locally

1. Clone the repository.
2. Open `index.html` in any browser.

## Notes

This is a frontend-only project (no backend, no JavaScript).
